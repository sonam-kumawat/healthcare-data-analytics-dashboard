# Healthcare Data Analytics Dashboard

## Overview
This project focuses on analyzing healthcare data to identify patient trends, hospital performance, billing patterns, and medical insights. SQL and Advanced Excel were used for data cleaning and analysis, while Power BI was used to create interactive dashboards.

---

## Objectives
- Clean and preprocess healthcare data
- Perform SQL-based analysis
- Analyze patient and billing trends
- Create interactive Power BI dashboards
- Generate business insights from healthcare data

---

## Dataset Summary
- Rows: ~55,000
- Columns: 17

### Key Features
- Patient demographics
- Medical conditions and treatments
- Admission details
- Hospital information
- Billing and insurance data

---

## Technologies Used
- SQL
- Advanced Excel
- Power BI

---

## Analysis Performed
- Overall patient and billing analysis
- Admission type analysis
- Medical condition analysis
- Hospital performance analysis
- Billing trend analysis

---

## Dashboard Features
- Patient distribution analysis
- Admission type comparison
- Billing amount trends
- Medical condition insights
- Hospital performance visualization

---

## Key Insights
- Identified high-cost treatment categories
- Compared hospital performance metrics
- Analyzed billing and admission trends
- Generated healthcare insights for decision-making
